# Flask Web Application

**Live Demo:** [www.note.infinityfreeapp.com](https://www.note.infinityfreeapp.com/login?next=%2F) 

This is a simple web application built with Flask, a micro web framework for Python. It provides a clean and modern user interface for managing notes.

## Features

* User authentication:
    * Secure user registration and login.
    * Session management for logged-in users.
    * Logout functionality.
* Note management:
    * Create new notes with titles and content.
    * View existing notes.
    * Delete notes.
* Modern UI:
    * Clean and visually appealing design.
    * Blurred backgrounds and subtle shadows for a contemporary look.
    * Smooth animations for a more engaging user experience.

## Technologies Used

* Flask: Python web framework.
* Bootstrap: Front-end framework for styling and responsiveness.
* Jinja2: Templating engine for dynamic HTML generation.
* HTML: Markup language for structuring the web pages.
* CSS: Styling language for visual presentation.
* JavaScript:  (Minimal use) For client-side interactions.

## Installation

1. Clone the repository: `git clone https://github.com/Sahuvivek12/Flask-web-application.git`
2. Install the dependencies: `pip install -r requirements.txt`
3. Run the application: `python main.py`

## Usage

1. Access the application in your web browser: `http://127.0.0.1:5000/`
2. Sign up for a new account or log in if you already have one.
3. Create, view, and delete notes.
